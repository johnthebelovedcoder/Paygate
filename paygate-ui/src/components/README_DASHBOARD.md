# Paygate Dashboard

## Overview
This is the main dashboard for the Paygate application, providing a comprehensive overview of your business performance.

## Features
- Revenue tracking with interactive charts
- Key performance metrics
- Recent activity feed
- Quick access to common actions
- Responsive design for all devices

## Accessing the Dashboard
The dashboard is the default page when you log into the Paygate application. Simply navigate to the root URL to access it.

## Using the Dashboard

### Time Range Filtering
Use the dropdown in the Revenue Summary section to filter data by:
- This Week
- This Month
- This Year
- Last Year
- Custom Range (select specific start and end dates)

### Quick Actions
The Quick Actions panel provides shortcuts to:
- Create a new paywall
- Upload content
- View customer list

### Interacting with Charts
Charts are powered by ApexCharts and support:
- Zooming
- Panning
- Downloading as images
- Tooltip information on hover

## Data Refresh
Dashboard data automatically refreshes when:
- The page loads
- Time range filters change
- You navigate back to the dashboard
